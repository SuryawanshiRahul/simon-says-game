Simon Says Game

An interactive memory game built using **HTML, CSS, and JavaScript**.
This project focuses on **event handling, DOM manipulation, and state logic**, forming a strong foundation for learning **React and React Native**.

---
What I Learned

 Event Handling

 Used `keydown` event to start the game
 Handled user clicks on buttons
 Managed multiple event listeners efficiently

DOM Manipulation

 Dynamically updated UI using JavaScript
 Changed styles (flash effects) using classes
 Updated text content for levels and game status

State Management (Core Concept)

 Managed game state using:

`gameseq` (system sequence)
 `userseq` (user input)
 Controlled game flow (start, level up, reset)

 Logic Building

 Implemented sequence matching logic
 Built level progression system
 Added game over detection

---
 Features

  Random color sequence generation
  Flash animation effects
  Level progression system
  Memory-based gameplay
  High score tracking using localStorage

---

  Tech Stack

 HTML
 CSS
 JavaScript (Vanilla JS)

---

 How to Play

1. Press any key to start the game
2. Watch the sequence carefully
3. Repeat the sequence by clicking buttons
4. Each level adds a new step
5. Game ends on wrong input
